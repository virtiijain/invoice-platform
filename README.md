## ūüôč FAQ

**Can I use this for free?**
Yes ‚ÄĒ the app is open source. Host it yourself using the free tiers of Vercel and Supabase.

**Is my data secure?**
All data is isolated per user via Supabase Row-Level Security. Nobody else can access your invoices.

**Does it support GST?**
Yes ‚ÄĒ client profiles support GST numbers and tax is auto-calculated in the invoice builder.
