<br/>

| Layer | Technology | Purpose |
|-------|-----------|---------|
| рҹ–јпёҸ **Framework** | Next.js 14 (App Router) | Full-stack React framework |
| рҹ”· **Language** | TypeScript | Type safety throughout |
| рҹҺЁ **Styling** | Tailwind CSS + shadcn/ui | UI components & design system |
| рҹ—„пёҸ **Database** | Supabase (PostgreSQL) | Data storage & real-time |
| рҹ”җ **Auth** | Supabase Auth | Email/password authentication |
| рҹ“„ **PDF** | react-pdf / jsPDF | Invoice PDF generation |
| рҹ“Ӣ **Forms** | React Hook Form + Zod | Form handling & validation |
| рҹ“Ҡ **Charts** | Recharts | Dashboard data visualization |
| рҹ—ғпёҸ **State** | Zustand | Client-side state management |
| рҹҡҖ **Deploy** | Vercel | Hosting & CI/CD |
